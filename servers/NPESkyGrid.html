SkyGridPE

SkyGridPE - An event server that was created as a part of NostalgiaPE3 Second Event on January 28th, 2023 at 8:05 PM (Moscow Time).
After the start of event, the server was relatively active until players discovered about the existence of banned world which also had SkyGrid map.
After the removal of banned world on January 31st, 2023(7:21PM Moscow Time), the server became anarchy. 
3 days later, on February 03rd, 2023 at 2:52PM, the server closed.

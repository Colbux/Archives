NostalgiaPE 1

NostalgiaPE was created in 2022/08/09 and had all original VanillaPE maps. 
The server had whitelist and required everyone to be verifed through discord to be able to join.
The server closed in 2022/09/07.

All NPE1 maps can be found on https://www.nostalgiaforum.xyz/index.php?threads/497/.
